# 🟩 Interactive Minecraft-Themed Portfolio Website

A modern, responsive, and interactive personal portfolio website inspired by Minecraft aesthetics and built using **HTML5, CSS3, and JavaScript**.

This project was developed as part of **Task 3** at **Maincraft Technologies**, focusing on JavaScript DOM manipulation, responsive UI design, and interactive web features.

---

## 🚀 Features

### ✨ Dynamic Navigation Bar

* Navbar changes style on scroll
* Active menu highlighting based on current section
* Smooth scrolling navigation

### 🌙 Dark Mode Toggle

* Switch between light and dark themes
* User preference saved using LocalStorage
* Instant UI transition effects

### 📩 Contact Form Validation

* Empty field validation
* Email format validation
* Inline error messages for better UX

### 🎮 Minecraft-Inspired Design

* Pixel-style branding
* Animated floating particle blocks
* Cinematic hero section background
* Modern gaming-inspired interface

### 📱 Fully Responsive

* Optimized for desktop, tablet, and mobile devices
* Flexible grid layouts
* Smooth animations and transitions

---
